# VoHive Web 中文术语表

本文档记录 Web 前端展示文案的统一术语。只约束面向用户的中文文案，不要求修改代码标识符、API 字段或第三方协议名。

| 场景 | 统一术语 | 说明 |
| --- | --- | --- |
| 本地出站能力 | 代理实例 | 用于 SOCKS5/HTTP 本地出站的可启停实例。按钮使用「新增实例」。 |
| VoWiFi 漫游上游 | 前置代理 | 用于 VoWiFi 分流到特定国家/地区的上游 Socks5 代理。按钮使用「新增前置代理」。 |
| SMS 列表与详情 | 会话 | 联系人维度的短信聚合入口统一称为「会话」。 |
| 设备公网地址刷新 | IP 轮换 | 统一不用「切换 IP」。 |
| eSIM | eSIM | 品牌/行业写法使用小写 e。 |
| 设备未运行状态 | 设备当前未运行 | 用于 AT/USSD 等依赖运行状态的不可用提示。 |
